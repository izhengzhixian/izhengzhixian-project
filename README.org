* z_tv
  这是在 linux 底下看视频不方便，发现了 you-get 以后觉得在看电视剧的时候，
  需要一个一个的去找每一集的网址，太不方便了，所以写了这么一个小程序，
  用来分析电视剧的信息，然后把每一集的地址保存到本地。

  目前实现了以下网址的分析:
  - youku
  - mgtv
  - bilibili

  具体功能可以直接运行 z_tv.py 来查看
